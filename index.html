Mapa de Cotação

Anexe os orçamentos em PDF dos fornecedores. A ferramenta lê cada um (mesmo escaneado), monta sozinha a lista de insumos a partir do que foi cotado, cruza os itens entre fornecedores e gera o mapa comparativo pronto pra baixar.

1 Orçamentos dos fornecedores 0

Solte quantos PDFs quiser — de uma vez ou aos poucos. Funciona com PDF de texto ou digitalizado (imagem). Não tem limite de fornecedores.

Só use isso se precisar forçar uma lista específica de itens (com Insumo / Unidade / Qtd. prevista) em vez de deixar a ferramenta montar a partir dos orçamentos.

2 Comparativo

Prévia editável — clique em qualquer valor pra corrigir, ou troque a correspondência no seletor de cada célula.

Este processamento roda no seu navegador: os PDFs viram imagem e são enviados para leitura via API da Anthropic só durante a análise, e o arquivo final é gerado e baixado localmente — nada fica salvo em servidor. Sempre confira os valores extraídos antes de usar o mapa para decisão de compra.
